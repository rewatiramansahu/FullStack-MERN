Hexa Decimal 4. rgb() 5. rgba() 6. hsl() 7. hsla() 8. linear-gradient() 9. radial-gradient() - rgb() : red, green, blue value range from 0 to 255 Syntax: { background-color: rgb(255,0,0); } - rgba() : red, green, blue & alpha [0 to 1] Syntax: { background-color:rgba(0,0,0,0.4); } - hsl() : hue, saturation, lightness hue is an angle from 0 to 360 deg. 0 to 120 => red 120 to 240 => green 240 to 360 => blue staturation is grey shade with 0 to 100% lightness is brightness with 0 to 100% - hsla() : alpha from 0 to 1 Syntax: { background-color: hsl(5deg, 80%, 60%); } { background-color: hsla(5def, 80%, 60%, 0.6); } - linear & radial gradient : you can set multiple colors only with background image. Syntax: body { background-image: linear-gradient(to top left, red 20%, yellow 70%, green); height: 100vh; } body { background-image: radial-gradient(red, yellow, green); height: 100vh; } CSS Inheritance ----------------------- - Inheritance in CSS is the process of configuring parent effects to child element. - Child element automatically implements some of parent features, which you can control by using following CSS values. a) inherit b) initial c) unset Syntax: { all : unset; => remove all default styles } Ex: Document
