============================== CSS Media Queries and Bootstrap ====================================== @Keyframes - Keyframe comprises of element with animation. - Keyframes require various phases a) Initial b) Final - Initial phase is defined using "from" - Final phase is defined using "to" - You can configure other phases from 1% to 100% Syntax: @keyframes name { from { } 50% { } to { } } - Keyframes are applied to element by using animation attributes animation-name animation-duration animation-delay animation-direction animation-iteration-count animation-timing-function Syntax: img { animation-name: keyframeName; animation-duration: 100ms; animation-direction:alternate; animation-iteration-count: 3 | infinite; } Ex: Document Ex: Document W E L C O M E Responsive Design - A responsive design can change the layout of page according to the viewport. - It provides fluid experince that fits the device screen. - To create responsive designs there are various implementations to follow 1. Meta Attributes 2. Responsive or Fluid Images 3. Responsive or Fluid Containers 4. Display Grid & Flex 5. Media Queries Meta Attribute: Fluid Images: Fluid Container: Display Grid & Flex: display:grid display:flex grid-row grid-column flex-wrap flex-row flex-column Media Queries: - Media have type : screen, print, speech - Media have orientation: landscape, portrait - Media have min-width & max-width. min-width : It defines value starting from specified units upto end. Syntax: min-width:600px => from 600px to end. max-width : It defines value starting from 0 to specified units. Syntax: max-width:600px => from 0px to 600px Bootstrap ------------- - It is an open source and cross platform component library used to build interactive UI faster. - There are various component libraries other than bootstrap a) Twitter Bootstrap b) Tailwind CSS c) Telerik d) DevExpress e) Mui f) jQuery UI etc.. - Open source allows to customize according to the requirements. - Cross platform runs on all OS services and Browsers. - Component is a combination of a) Markup HTML b) Styles CSS c) Logic JavaScript | jQuery - A component have presentation, styles and functionality ex: navbar carousel cards accordion etc..
