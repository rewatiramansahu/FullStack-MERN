========================= CSS Display Grid and Background ========================= Flex Wrap - It can control wrapping of elements in container. - If set to "nowrap" elements will adjust according to width of container. - If set to "wrap" elements will span to multiple rows. Syntax: .container { display : flex; flex-wrap: nowrap | wrap; } Ex: Document
Display Grid: - It provides attributes to create a responsive layout. - You can re-arrange the content in page dynamically according to media orientation or size. a) display:grid b) grid-row c) grid-column d) grid-gap e) grid-template-columns - A responsive grid have max 12 columns [1200 px] - You can set size of columns in "fr" [fractions] or in "px" [pixles]. - Size in "fr" will be responsive.
