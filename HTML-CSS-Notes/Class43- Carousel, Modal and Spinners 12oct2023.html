==================================== Carousel, Modal and Spinners ============================================== Components 1. alerts 2. badge 3. card 4. pagination Carousel - It is used to create sliding and fading banners. - You can create a slide show of content, which can be controlled manually and with automation. Basic Carousel Classes: .carousel .carousel-inner .carousel-item .carousel-fade .slide Syntax:
...... your content....
... add multiple items...
- Carousel can't display any item directly. - Carousel needs any one item set to active. [only one item must be active]
- Carousel can't start animation automatically. - To start animation you have to define the attribute "data-bs-ride" for carousel.
- Carousel doesn't have any default transition, you have to set transition using the classes a) slide b) carousel-fade
Ex: Home
Carousel Controls: - You can add controls for carousel so that user can navigate to previous and next slide. Classes: .carousel-control-prev .carousel-control-prev-icon .carousel-control-next .carousel-control-next-icon .carousel-dark - Carousel controls are usually buttons. Syntax: - Carousel controls need jQuery attributes to handle navigation. data-bs-slide="prev" data-bs-slide="next" data-bs-target="#carousel_id" Ex: Home
Carousel Indicators: - Indicators allow to navigate random. - You can navigate to any slide directly by using indicators. Class: .carousel-indicators - Usually indicators is a collection of buttons or list items. - Indicators require jquery attribute "data-bs-slide-to=indexNumber" - Index number of items will start with "0".
Ex: Home
- Carousel item have default timing, you can set custom interval by using the attribute "data-bs-interval=milliSeconds" Syntax:
Ex: Home
- You can add any content into carousel item, which includes text, images, tables, forms, links etc.. Ex: Home
