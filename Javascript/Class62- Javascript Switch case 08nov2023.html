Class62- Date 08nov2023 =============================== javascript Switch Case =============================== Switch Selector: - Switch is used to interrupt the flow of electrons in circuit. - Switches are classified into various types a) Toggle switch b) Push button switch c) Selector switch d) Joystick switch etc.. - Selector switch allows to choose exactly the block that matches given condition. Syntax: switch(value | expression) { case value | expression: statements on match; jump_statement; default: statements when value | expression doesn't match; jump_statement; } Ex: FAQ's: 1. Can we define switch without default ? A. Yes. 2. Can we define default above or between cases? A. Yes. 3. Can we use case without "jump" statement? [break] A. Yes. If jump statement is not defined then it will stop when next jump occurs. Ex: 4. Can we use "return" as jump statement in switch? A. Yes. 5. What is difference between break and return? A. break will terminate the current block but will stay in function, compiler is ON. return will terminate the compiling. [stop every thing] Note: "return" can be used only with in function. Ex: 6. Can we define case value as string or boolean? A. Yes. Ex: 7. Can we define multiple cases for single block? A. Yes. Ex: Note: You can transform and change case if you are using a string as case value. Ex: 8. How to define a case and switch for range of values? A.
