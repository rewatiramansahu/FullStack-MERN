=========== CSS Positions and Z Index =========== CSS Positions 1. Static 2. Absolute 3. Fixed 4. Relative 5. Sticky Static - It keeps element according to normal flow of document. - It will not allow to change the position of element. - It is not effected by left, right, top and bottom placement if defined. Absolute - It removes element from normal flow of document. - It allows to change the position using left, right, top and bottom values. - It is attached to content in page. - It moves along with content in page. Fixed - It removes element from normal flow of document. - It allows to change position using left, right, top and bottom. - It is attached to browser not to content. - It locks scrolling. Sticky - It keeps element according to normal flow. - It will not allow to change position. - It can keep element sticky after reaching specific point on page while scrolling. Ex: Document
