Alerts - Alert is an embedded message box in page. - You can dynamically control using client side script. Classes: .alert .alert-dismissible .alert-{contextual} primary, secondary, danger, success, warning, info, dark etc.. .alert-title .alert-text .alert-link jQuery Attributes: - Bootstrap component use jquery attributes to configure and handle functionality. - Bootstrap 5 version defines the attributes using "data-bs-{ }" - Bootstrap upto 4 version defines the attributes using "data-{ }" - Bootstrap jquery attributes are various types a) data-bs-target b) data-bs-toggle c) data-bs-dismiss d) data-bs-slide e) data-bs-ride etc... - To close any component in bootstrap you can attach the attribute "data-bs-dismiss" to any clickable element.
