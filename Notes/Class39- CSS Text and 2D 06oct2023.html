CSS 2D - 2D refers to 2 dimensional effects. - You can change the orientation and position of element along X & Y axis. - You can configure by using "transform" attribute. - Transform methods for 2D are translate() scale() skew() rotate() matrix() Translate: - It can change the position of element along X and Y axis. translate(x, y) translateX() translateY() Ex: Document Scale : - It can change the size of element. scale() both width and height scaleX() width scaleY() height - Value of scale 1 = 100% Ex: Document Ex: Document
Skew: - It is used to tilt element by specified angle. skew(degx, degy) skewX() skewY() Ex: Document
Rotate: - It rotate element by specified angle. rotate(degx, degy) => always use only one angle in short hand rotateX() rotateY() Ex: Document
Note: You can't configure multiple transforms for element, only the latest transform will apply. You can define matrix(), which allows few transforms simultaneously at the same time. Syntax: matrix(scaleX(), skewX(), skewY(), scaleY(), translateX(), translateY())
