Class65- Date 11nov2023 ============================= JavaScript Arrays =================================== - ES5+ versions introduced Array destructuring technique. Syntax: Without Destructure var products = ["Samsung TV - 45000.55", "Fastrack Watch - 2400.45"]; var tv = products[0]; var watch = products[1]; Syntax: Using Destructure var products = ["Samsung TV - 45000.55", "Fastrack Watch - 2400.45"]; var [tv, watch] = products; var [tv, watch, mobile] = products; mobile = undefined; - JavaScript array can store any type of value, primitive or non-primitive.
