######### Tables in HTML ###### - Tables are used to organize data into rows and columns. - In modern web applications tables are used for designing "Grid" components. - Grid is a component that allows user to browse, edit, insert, update, delete, filter, group, sort data etc. - Grid is designed by using table in HTML. - HTML 5 provides following elements for table. Table Attributes: 1. Frame, Rules & Border frame : It defines a frame for entire table. It uses the values "box, above, below, rhs, lhs, void" rules : It defines rules for rows, columns and groups. It uses the values "all, rows, cols, groups, none"
border : It defines a border for table cell. It requires rules to be removed from table. You can't set border if rules attribute is defined. It uses 0 or 1 as value. 0 : no border [false] 1 : border [true]
2. Cellspacing and Cellpadding cellspacing : It sets space between cells. cellpadding : It sets space between border and cell content.
