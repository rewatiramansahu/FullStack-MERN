Adding Elements into Array: push() : It adds new element(s) as last element(s). unshift() : It adds new element(s) as first element(s). splice() : It adds new element(s) at specific position. Syntax: arrayName.push("element1", "element2",...) arrayName.unshift("e1", "e2", ...) arrayName.splice(indexNumber, deleteCount, "element1", "element2", ...) Ex: var categories = ["All", "Electronics"]; categories.push("Men's"); categories.splice(1,0,"Women");
