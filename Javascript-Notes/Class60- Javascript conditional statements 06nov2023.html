Forward Jump 2. Simple Decision 3. Multi Level Decisions 4. Multiple Decisions Forward Jump: - It is a decision making approach where there is no alternative. - User will go to next only when the current condition is satisfied. Syntax: if (condition) { statement on true; } Ex: Document
