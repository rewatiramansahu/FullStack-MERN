============== CSS Display =================== 1. None 2. Block 3. Inline 4. Inline-Block 5. Flex 6. Grid None: - It hides element in page. - It removes the allocated space for element. Syntax: img { display: none; } FAQ's: What is difference between "display:none" and "visibility:hidden" ? Ans : display:none => removes the allocated space visibility:hidden => keeps the allocated space Block & Inline: - Block can display element with a line break. - Inline can display element without line break. - You can use both for displaying hidden element. Syntax: span { => display in new line display:block; } span { => display in the same line display:inline; } Ex: Document

Ans: - Flex is used only for container that comprises of elements. - It can align elements flexible horizontally & vertically. - Inline is used for individual element to keep along with other elements. - It is not flexible for alignments.
