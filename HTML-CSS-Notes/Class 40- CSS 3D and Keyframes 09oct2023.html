=============================== CSS 3D and Keyframes ================================== CSS 2D Matrix: - It configures multiple transforms to element. - The matrix pattern comprises of scale, skew & translate. Syntax: { transform:matrix(scaleX(), skewX(), skewY(), scaleY(), translateX(), translateY()) } Scale : 1 = 100% Skew : 0 to 1 [0.3 = 30deg] translate : 100 = 100px Ex: Document Ex: Document CSS 3D Transforms - 3D refers to 3 dimensional transforms. - 3D comprises of X, Y & Z. - 3D transform methods are similar to 2D translate3D() scale3D() skew3D() roate3D() matrix3D() Ex: Document CSS Transition - Transitions are used to control the transformation. transition : short hand transition-duration : total time taken for transformation transition-delay : time taken to start the transformation transition-property : it refers to the property that uses duration [time] transition-animation-function : It applies pre-defined animations EX: Document
CSS Keyframes & Animations - Keyframe is used to create animation for any element in page. - A keyframe comprises of element with any specified effects like size, position, orientaiton, color etc. - A good basic animation comprises of 30 frames per second. [30fps] - CSS can configure "Tweening" animation, which comprises of initial and final state. Syntax: @keyframes name { from { initial_state; } to { final_state; } } - Keyframe is applied to any element in page by using animation attributes animation-name animation-duration animation-direction animation-iteration-count animation-delay animation-timing-function Ex: Document Ex: Document
