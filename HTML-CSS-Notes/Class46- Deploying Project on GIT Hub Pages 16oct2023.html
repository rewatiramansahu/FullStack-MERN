=============================== Deploying Project on GIT Hub Pages =============================== Deploying Project on Cloud - GIT Hub Pages - Firebase - Azure - AWS etc.. Deploying on GIT 1. Create a new GIT account https://github.com/ 2. Login into GIT account 3. Click "Create - New Repository" [new button] Repository Name : ui-portfolio Description : anything about this project Access Type : public Add a Readme File: select checkbox Create Respository 4. Download and Install "GIT Desktop" https://desktop.github.com/ 5. Open GIT Desktop from your programs & Login into GIT desktop 6. Go to File menu => Clone Repository 7. Select your repository name and set physical path repostory name : ui-portfolio physical path : d:\ui-portfolio 8. Click "Clone" button 9. Open Visual Studio Code => open your project folder d:\ui-portfolio\ui-portfolio 10. Add package.json >npm init -y 11. Add index.html in project folder 12. Add folders a) public => keep all other HTML files, images, documents b) src => styles and scripts Note : You have to use CDN links for libraries. 13. Go to Index.html and link bootstrap-icons & bootstrap CDN 14. Design your web page with bootstrap css and javascript. UI-Portfolio
